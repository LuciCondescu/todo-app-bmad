Todo App — Design Direction A: "Canvas"

Clinical minimalism. Cool neutrals, blue accent used only for interactive affordances. System fonts. One-column layout. Monochrome foreground, single accent, single danger color. Mockups below render the same tokens specified in the UX design specification.

Legend

  • Mobile frames are 320px wide (minimum supported viewport).
  • Desktop frames render at the ~640px app container width.
  • Every interactive element is ≥ 44×44px (tap-target minimum).
  • Completed text is #1A1A1A at 60% opacity → ~6.8:1 contrast against #FAFAFA (WCAG AA).
